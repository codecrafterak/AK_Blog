Learn EmailJS Contact Forms

Step-by-step guidance to build modern forms with golden glassmorphism styling.

Get Started See Examples

Overview

Goal

Create a working EmailJS contact form styled with golden glassmorphism.

What you'll learn

Setup, build, connect, and troubleshoot your form step by step.

Prerequisites

EmailJS Account

Create and log in. Add a service (e.g., Gmail).

Template

Include variables: {{name}}, {{email}}, {{message}}.

Public Key

Copy your EmailJS public key from the dashboard.

EmailJS Setup

Add SDK

Include the EmailJS browser SDK before your script.

Initialize

Call emailjs.init('YOUR_PUBLIC_KEY').

IDs

Note your service ID and template ID.

Build the Form

Structure

Use id="form" and inputs named "name", "email", "message".

Accessibility

Labels with for=, required fields, focus styles.

Button State

Show sending/sent feedback and disable to prevent duplicates.

Connect the Script

<script>
const SERVICE_ID = 'default_service';
const TEMPLATE_ID = 'template_qhe68wc';
const PUBLIC_KEY = 'YOUR_PUBLIC_KEY';

emailjs.init(PUBLIC_KEY);

const form = document.getElementById('form');
const btn = document.getElementById('button');

form.addEventListener('submit', (e) => {
  e.preventDefault();
  btn.disabled = true;
  const original = btn.textContent;
  btn.textContent = 'Sending...';

  emailjs.sendForm(SERVICE_ID, TEMPLATE_ID, form)
    .then(() => {
      btn.textContent = 'Sent!';
      setTimeout(() => { btn.textContent = original; btn.disabled = false; }, 1200);
      form.reset();
      alert('Message sent successfully!');
    })
    .catch((err) => {
      console.error(err);
      btn.textContent = original;
      btn.disabled = false;
      alert('Error sending message.');
    });
});
</script>

Test & Troubleshoot

Verify IDs

Check service ID, template ID, and public key.

Match Fields

Template variables must match input names.

Email Service

Confirm emails aren’t blocked or flagged as spam.

Debug

Use console.error for details. Add CAPTCHA for spam protection.
